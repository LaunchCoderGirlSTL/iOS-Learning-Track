My Split View / Page View Controller
====================================

Objectives
----------

   - Become familiar with split view controller and page view controller delegates.

   - Become comfortable with incorporating different controllers to a split view controller

Requirements
------------

   - Create a project with the split view controller as the parent.

   - Create a master view controller with at least 10 items

   - Create a secondary view controller that is a page view controller with at least 3 different pages

   - Should be able to swipe back and forth between the pages. If you swipe past the last page, it should take you back to the first page and if you swipe backwards from the first page, it should bring you to the last page.
